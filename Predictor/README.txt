Directions to run the project:
1. Unzip gl2010_19.zip (download link: https://www.retrosheet.org/gamelogs/gl2010_19.zip) in data folder.
2. Run combining_all_text_files.ipynb in the dataset_generator folder.
3. Run MLB_scraper_2.ipynb in the dataset_generator folder.
5. Run all_data_eda.ipynb in the dataset_generator folder.
6. Run feature_engineering.ipynb (this is the original researcher's ramdom forest analysis file) or run feat_eng_xgb.ipynb (this is our modified xgboost analysis file)
